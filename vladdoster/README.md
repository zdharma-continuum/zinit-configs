# [vladdoster](https://github.com/vladdoster) zinit configuration

## Features

- System agnostic (macOS or Linux)
- Neovim nightly
- `pip` auto-complete
- `fzf`
- fast (~200 ms on most systems)
- GNU utility for macOS
